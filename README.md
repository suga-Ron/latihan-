# Sistem Manajemen Gudang (Warehouse Management System)

Aplikasi sederhana berbasis CLI (*Command Line Interface*) yang dibuat dengan Python untuk mengelola inventaris barang di gudang.

## Fitur
- [x] Menambah barang baru
- [x] Melihat daftar barang
- [x] Menghapus barang
- [x] Penyimpanan data permanen menggunakan JSON

## Cara Menjalankan
1. Pastikan Anda sudah menginstal Python di komputer.
2. Unduh atau *clone* repositori ini.
3. Buka terminal/command prompt.
4. Masuk ke folder proyek.
5. Jalankan perintah:
   ```bash
   python latihan.py
